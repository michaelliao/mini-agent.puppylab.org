# Mini-Agent

Mini-Agent is a personal AI assistant that runs on your own computer.

It provides:

- Multi-task execution: Run multiple tasks with AI.
- Skills: Skills described in Markdown documents.
- Session Management: Isolated conversation contexts per task.

# Install Mini-Agent

Get Mini-Agent by git clone command:

```bash
$ git clone https://github.com/michaelliao/mini-agent.puppylab.org.git
```

Install dependencies:

```bash
$ sudo apt install python3-openai python3-prompt-toolkit
```

# Run Mini-Agent

```bash
$ cd path/to/mini-agent.puppylab.org
$ cd mini-agent
$ ./agent.py
```
